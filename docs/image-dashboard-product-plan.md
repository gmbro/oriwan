# 이미지 기반 러닝 인증 대시보드 제품 기획

작성일: 2026-05-05

## 목표

운영자가 참가자를 직접 등록하고 러닝 인증 이미지를 업로드하면, AI가 이미지 속 텍스트만 추출해 날짜, 이름, 거리, 시간을 구조화한다. 누락된 값은 운영자가 직접 보정하고, 인증 여부와 향상도를 대시보드에서 확인한다.

## 보완된 요구사항 반영

| 요구 | 반영 방식 |
| --- | --- |
| 인원은 운영자가 직접 추가 | 대시보드 참가자 추가 영역 |
| 이름도 운영자가 입력 | 참가자 이름/닉네임 저장 |
| 이미지에서 날짜 자동 추출 | OCR 분석 prompt와 `record_date` 구조화 |
| 날짜가 없으면 수동 등록 | 업로드 기본 날짜 임시 적용 + 확인 필요 상태 |
| 배경이 보이면 제거하고 텍스트만 추출 | AI prompt에서 배경/앱 UI 무시, 텍스트/숫자만 추출 지시 |
| 인증 여부 시계열 | 최근 14일 참가자 x 날짜 매트릭스 |
| 랭킹 | 인증 횟수, 누적 거리, 누적 시간 정렬 |
| 시간이 없으면 수동 입력 | 검수 테이블과 수동 기록 입력 폼 |

## 핵심 화면

- 요약 카드: 참가자 수, 선택일 인증률, 확인 필요, 거리, 시간
- 이미지 업로드: 여러 장 업로드, 기본 날짜 선택, AI 분석
- 참가자 등록: 이름/닉네임 추가
- 수동 기록 입력: 참가자, 날짜, 거리, 시간 입력
- 인증 시계열: 최근 14일 인증/확인필요/미제출 표시
- 랭킹: 인증/거리/시간 기준 전환
- 개인 그래프: 거리와 시간 추이
- 기록 검수: AI 추출값 수동 수정 및 인증 처리

## 상태 판정

- `certified`: 참가자, 날짜, 거리, 시간이 모두 채워진 경우
- `needs_review`: 날짜가 이미지에 없거나, 시간이 없거나, 참가자 매칭이 애매한 경우
- `rejected`: 운영자가 반려한 경우
- `missing`: 시계열에서 제출 기록이 없는 경우

## 향후 개선

1. 중복 이미지 해시 감지
2. OCR 결과 원본 이미지 미리보기
3. 참가자별 상세 페이지
4. 월간/주간 리포트
5. CSV 내보내기
6. 카카오톡/슬랙 제출 이미지 자동 수집
