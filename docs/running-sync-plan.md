# 이미지 기반 러닝 인증 운영 기획안

작성일: 2026-05-05

## 제품 결정

러닝 기록 연동 기능은 제거한다. 스내사 3기 대시보드는 운영자가 업로드한 러닝 인증 이미지에서 AI가 날짜, 이름, 거리, 시간, 페이스를 추출하고, 운영자가 검수해 인증 현황과 향상도를 보는 대시보드 제품으로 운영한다.

## 데이터 추출 정책

| 항목 | 자동 추출 | 누락 시 처리 |
| --- | --- | --- |
| 참가자 이름 | 이미지 텍스트와 등록 이름 매칭 | 검수 테이블에서 수동 선택 |
| 날짜 | 이미지에 보이는 날짜 추출 | 업로드 기본 날짜 임시 적용 후 확인 필요 |
| 거리 | km 숫자 추출 | 수동 입력 |
| 시간 | HH:MM:SS 또는 MM:SS 추출 | 수동 입력 |
| 페이스 | 이미지에서 추출하거나 거리/시간으로 계산 | 거리/시간 입력 후 자동 계산 |
| 앱 출처 | 이미지에 보이면 추출 | 비워둠 |

## 인증 상태

- `certified`: 참가자, 날짜, 거리, 시간이 모두 확인된 상태
- `needs_review`: 날짜/시간/참가자 매칭이 없거나 애매한 상태
- `rejected`: 운영자가 반려한 상태
- `missing`: 시계열 표시용 미제출 상태

## 대시보드

- 참가자 수
- 선택일 인증률
- 확인 필요 건수
- 선택일 누적 거리
- 선택일 누적 시간
- 이미지 업로드/OCR 분석
- 수동 기록 입력
- 최근 14일 인증 시계열
- 인증/거리/시간 랭킹
- 개인별 거리/시간 그래프
- 기록 검수 테이블
